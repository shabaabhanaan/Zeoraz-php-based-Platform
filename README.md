# Zeoraz - Multi-Vendor Market

A professional multi-vendor marketplace application built with PHP and MySQL.

## Features
- User Authentication (Login/Register)
- Multi-Vendor Support (Seller Dashboard)
- Product Management (Add/Edit/Delete)
- Shopping Cart and Checkout Flow
- Order History and Tracking

## Tech Stack
- **Backend:** PHP
- **Database:** MySQL
- **Styling:** CSS (Maintained from original designs)

## Setup and Installation (Local Development)

1. **Prerequisites:**
   - XAMPP or similar local PHP server.
   - MySQL database.

2. **Database Setup:**
   - Create a database named `multi_vendor_market` (or as configured in `includes/db.php`).
   - Import the `database.sql` file provided in the root directory.

3. **Application Setup:**
   - Clone or copy this project into your `htdocs` folder (e.g., `C:\xampp\htdocs\multi-vendor-market`).
   - Update `includes/db.php` with your database credentials.

4. **Running the App:**
   - Start Apache and MySQL in XAMPP.
   - Navigate to `http://localhost/multi-vendor-market/` in your browser.

## Deployment
This project is optimized for deployment on PHP-based hosting environments like **Infinity Free**.
